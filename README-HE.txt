מתקדמים v60.2 – תיקון שמירת כושר ותמונות

מה להעלות ל-GitHub:
יש להעלות את כל התוכן שבתיקייה UPLOAD-TO-GITHUB-ROOT לשורש המאגר.
חשוב במיוחד להעלות גם את תיקיית assets בשלמותה. בלי התיקייה התמונות של התרגילים לא יוצגו.

בדיקה לאחר העלאה:
1. המתינו לסיום הפרסום של GitHub Pages.
2. פתחו את האתר ולחצו Ctrl+F5.
3. התחברו עם Google.
4. בחרו מסלול כושר ולחצו שמירה. המתינו להודעה: מסלול הכושר נשמר בענן ✓.
5. עברו לאנגלית וחזרו לכושר.
6. בדקו שהתמונות מוצגות.

אם מופיעה הודעה האיור לא הועלה לאתר, סימן שתיקיית assets או חלק ממנה לא הועלו ל-GitHub.

מתקדמים v50.6

מה תוקן:
1. שמירת הגדרות הכושר נשמרת בכמה מפתחות גיבוי.
2. הנתונים נשמרים גם בעת מעבר לטאב אחר, סגירת הדף או יציאה מהאפליקציה.
3. נוספה הכנה למעקב כניסות דרך Firebase.
4. נוספה אפליקציית חדר בקרה נפרדת בתיקייה mitkadmim-control-room.

חשוב:
כתובת file:/// היא קובץ מקומי בלבד. אנשים אחרים אינם יכולים להיכנס אליה, ולכן אי אפשר לעקוב אחרי מבקרים דרך כתובת כזו.
כדי להפעיל מעקב אמיתי, יש להעלות את האפליקציה ל-GitHub Pages ולחבר Firebase.
יש להדביק את אותם פרטי Firebase בשני קבצים:
- mitkadmim-v50-final-entry-design/c1/firebase-config.js
- mitkadmim-control-room/firebase-config.js

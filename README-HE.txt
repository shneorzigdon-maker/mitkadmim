מתקדמים v60.3 — גרסה נקייה

יש להעלות את כל הקבצים והתיקיות שבתיקייה זו ישירות לשורש מאגר GitHub ולהחליף את הקבצים הישנים.
אין בתיקייה גרסאות כפולות, קובצי תצוגה מקדימה או חדר בקרה לא מוכן.

בדיקה:
1. להמתין לסיום הפרסום של GitHub Pages.
2. לפתוח את האתר וללחוץ Ctrl+F5.
3. להתחבר עם Google.
4. לבחור מסלול כושר וללחוץ שמירה.
5. לוודא שמופיעה ההודעה: מסלול הכושר נשמר בענן ✓
6. לעבור לאנגלית ולחזור לכושר.

בגרסה זו מסלול הכושר נשמר בשני מקומות באותו מסמך Firestore:
state.fitness וגם fitness כשדה עצמאי, ולאחר השמירה מתבצעת קריאה חוזרת לאימות.
